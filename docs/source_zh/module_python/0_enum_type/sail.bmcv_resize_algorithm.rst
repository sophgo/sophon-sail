sail.bmcv_resize_algorithm
___________________________


定义图像resize中常见的插值策略


**接口形式:**
    .. code-block:: python

        sail.bmcv_resize_algorithm.BMCV_INTER_NEAREST
        sail.bmcv_resize_algorithm.BMCV_INTER_LINEAR
        sail.bmcv_resize_algorithm.BMCV_INTER_BICUBIC

**参数说明**

* sail.bmcv_resize_algorithm.BMCV_INTER_NEAREST

最近邻插值算法

* sail.bmcv_resize_algorithm.BMCV_INTER_LINEAR

双线性插值算法

* sail.bmcv_resize_algorithm.BMCV_INTER_BICUBIC

双三次插值算法
